-----------------------------------------------------------------------------------------------------------------------------
📦 Підготовка файлів
-----------------------------------------------------------------------------------------------------------------------------

Скачайте архів з ботом (файли bot.py, requirements.txt, startup.bat, install.bat, README.txt)

Розпакуйте всі файли у папку C:\Dtek_bot (створіть її вручну, якщо потрібно).

-----------------------------------------------------------------------------------------------------------------------------
⚙️ Установка необхідних компонентів
-----------------------------------------------------------------------------------------------------------------------------

Відкрийте папку C:\Dtek_bot.

Двічі клацніть на файл install.bat — він автоматично встановить все необхідне:

Python

Playwright

Бібліотеки для Telegram

Після завершення установки з'явиться повідомлення "Установка завершена".

-----------------------------------------------------------------------------------------------------------------------------
🚀 Запуск бота вручну
-----------------------------------------------------------------------------------------------------------------------------

У тій самій папці C:\Dtek_bot двічі клацніть на файл startup.bat.

Відкриється чорне вікно — бот запущено і він очікує команди.

Згорніть його - якщо закрити це вікно, вимкнеться бот.

-----------------------------------------------------------------------------------------------------------------------------
🔁 Налаштування автозапуску при включенні комп’ютера
-----------------------------------------------------------------------------------------------------------------------------

Натисніть Win + R, введіть shell:startup і натисніть Enter.

Відкриється папка автозапуску.

У папці C:\Dtek_bot:

Клацніть правою кнопкою на startup.bat

Оберіть "Створити ярлик"

Перетягніть цей ярлик у папку автозапуску

Тепер бот буде запускатися автоматично при кожному включенні комп’ютера.

-----------------------------------------------------------------------------------------------------------------------------
📲 Користування ботом у Telegram
-----------------------------------------------------------------------------------------------------------------------------

Відкрийте Telegram.

Перейдіть за посиланням: https://t.me/address_your_bot

Натисніть "Start" або "Почати".

-----------------------------------------------------------------------------------------------------------------------------
📋 Команди бота
-----------------------------------------------------------------------------------------------------------------------------

/addaddress <місто> <вулиця> <будинок> — додати адресу для перевірки (напр.: Дніпро Космічна 19)

/listaddresses — переглянути список збережених адрес

/deleteaddress <номер> — видалити адресу з списку

/status <номер> — перевірити статус для конкретної адреси (напр.: /status 1)

/status all — перевірити всі адреси

/track <номер> — включити автоматичне відстеження змін

/stoptrack <номер> — зупинити відстеження

-----------------------------------------------------------------------------------------------------------------------------
🛠️ Файли, які створюються автоматично
-----------------------------------------------------------------------------------------------------------------------------

addresses.json — список ваших адрес

tracking.json — які адреси відстежуються

tracking.log — журнал змін

startup.log — журнал запуску

schedule.XXXX.png — скріншоти графіків

-----------------------------------------------------------------------------------------------------------------------------
✅ Готово!
-----------------------------------------------------------------------------------------------------------------------------

Бот працює у фоновому режимі, перевіряє графіки відключень і надсилає повідомлення при змінах.

Якщо щось не працює — просто перезапустіть startup.bat або зверніться до технічної підтримки.
